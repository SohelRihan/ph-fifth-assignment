Important FAQ

What is the Document Object Modal(DOM)?

The Document Object Model (DOM) is a programming interface for web documents.

How do you select an element from the DOM?

Use getElementById() for unique IDs, Use getElementsByClassName() and getElementsByTagName() for collections of elements etc.

What is event delegation in the context of the DOM and why is it useful?

Event delegation is a technique in JavaScript where a single event listener is added to a parent element to manage events for multiple child elements.

How do you manipulate an element's attribute and styles using DOM?

Use getAttribute(), setAttribute(), and removeAttribute() to manipulate attributes of elements & Styles: Use the style property to set, get, or remove inline styles of elements
